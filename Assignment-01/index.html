Introduction

The purpose of this long single-file HTML page is to showcase many different HTML elements and their typical usage. Below you'll find headings, paragraphs, lists, images, video, tables, forms, and more.

Sub note

Use semantic tags to improve accessibility and document structure. For example, <article>, <section>, <nav>, and <footer> are semantic. Non-semantic tags like <div> and <span> are useful for generic grouping.

Quick Facts

  • HTML stands for HyperText Markup Language.
  • HTML5 introduced many semantic elements.
  • Self-closing tags used here: <br />, <hr />, <img />, <input />.
